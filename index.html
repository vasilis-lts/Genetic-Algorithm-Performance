The genetic algorithm generates an array (Generation) of random words / phrases, like the brute force algorithm, but it also evaluates each element, based on how similar it is to the target word / phrase (fitness calculation).

Then it takes all the more (fit) similar to the target elements and combines them, which results in creating new children arrays that look more and more like the target.

This is repeated until an element identical to the target is created in one of those arrays.
